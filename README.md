# Authbox

An OAuth 2 provider that allows you to specify authorised users of multiple other OAuth providers.

### Prerequisites

- Ruby 2.4.x + bundler
- PostgreSQL
- GitHub Application

## Setup

```sh
$ git clone git@github.com:adambutler/authbox.git
$ cd authbox
$ cp .env.example .env
$ rake db:setup
```
